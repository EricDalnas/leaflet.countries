Manual Label Placement Demo

Feature Overview

This demo shows how to manually place labels for regions/countries using the labelLatLng property. This is useful for countries with scattered territories (like Spain) where the default center-based label placement would place the label over a territory different from the main region.

How to Use Manual Label Placement

Add a labelLatLng property to your country/region data object. The property accepts either an array [lat, lng] or an object {lat, lng}: 

var layer = L.countriesLayer({
  geoJsonUrl: '...',
  data: {
    US: { 
      color: '#0047ff',
      labelLatLng: [39.8283, -98.5795]  // Manually place label at Kansas
    },
    ES: {
      color: '#d40000',
      labelLatLng: { lat: 39.5, lng: -3.7 }  // Manually place label over Spain mainland
    }
  }
}).addTo(map);

Note: The label will be placed at the specified coordinates instead of at the center of the feature's geometry. This allows precise control over label positioning.
